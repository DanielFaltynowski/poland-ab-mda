# 🇵🇱 Polska A/B, czyli analiza wielowymiarowa dualizmu społeczno-ekonomicznego regionów Polski

Inny pomysł na nazwę: Dwie Polski

Niniejsze repozytorium zawiera projekt zaliczeniowy przygotowany na potrzeby przedmiotu **Analiza wielowymiarowa**. Kurs koncentrował się na analizie danych regionalnych w Polsce, badając zależności między wybranymi zmiennymi a populacją poszczególnych województw Polski.

Choć punktem wyjścia była klasyczna koncepcja dualizmu Polski, analiza danych szybko wykazała, że podział na A i B jest znacznie bardziej złożony, niż zakładaliśmy. Skala odkrytych uwarunkowań sprawiła, że projekt ewoluował w kompleksowe studium różnic regionalnych, szczegółowo mapujące dysproporcje między województwami. Kluczowe wnioski oraz opis przebiegu prac zamieszczamy poniżej, natomiast szczegółowe dane, wykresy i pełną analizę znajdziesz w załączonym raporcie (plik...), do którego lektury gorąco zachęcam.
